Зависимости:
	Python 2.7.6

1. Задача на минимальное расстояние.
Дан набор из N точек на плоскости (для простоты можно считать,
что у всех точек целочисленные координаты).
Найдите минимальное расстояние между двумя точками из этого набора.

Запуск:
	python distance.py
Ввод данных:
	a) После вопроса "Какое количество точек Вы хотите рассчитать?" необходимо ввести
		одно натуральное число больше 1, нажать "Enter".
	б) Затем через пробелы введите две координаты для первой точки, нажмите "Enter".
	в) Введите координаты для второй точки, нажмите "Enter", затем повторите ввод
		для последующих точек.
	г) При правильном вводе данных будет выведен результат - минимальное расстояние
		между двумя точками из введенного набора.
	д) Если данные будут введены с ошибкой, будет запрошен повтор ввода.
	
2. Задача по переводу дроби в k-ичную систему координат.
Даны два числа: a и b. Найдите значение числа a/b, записанного в k-ичной системе
счисления. Если a/b — периодическая дробь, то период следует заключить в скобки.

Запуск:
	python conv2base.py
Ввод данных:
	a) После вопроса u"Введите делимое, делитель и основание для новой системы
		счисления через пробел:" необходимо ввести три числа,третье должно быть
		не меньше 2 и не больше 36, затем нажать "Enter".
	б) Будет выведен результат расчетов.
	в) Если данные будут введены с ошибкой, будет запрошен повтор ввода.
